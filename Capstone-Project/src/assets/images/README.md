# Logo Files

Silakan tambahkan file logo berikut ke folder ini:

1. **dicoding-logo.png** - Logo Dicoding (format PNG dengan background transparan atau putih)
2. **capstone-logo.png** - Logo Capstone dengan huruf "g" (format PNG dengan background transparan atau putih)

## Catatan:
- Pastikan nama file sesuai dengan yang disebutkan di atas
- Format yang disarankan: PNG dengan background transparan
- Ukuran yang disarankan:
  - Dicoding logo: tinggi sekitar 50px (akan di-scale otomatis)
  - Capstone logo: 60x60px (akan di-scale otomatis)

Logo akan ditampilkan di hero section landing page dengan styling yang menarik.
